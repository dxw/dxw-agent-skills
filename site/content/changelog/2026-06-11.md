Bug in 11ty `addWatchTarget()` means that adding an external watch target in Eleventy config (in this case, the `/.agents/` root folder) causes stale rebuild behaviour.

---+---

Diff:

```diff
-  eleventyConfig.addWatchTarget("../.agents/");
+  eleventyConfig.addWatchTarget("./../.agents/");
```
